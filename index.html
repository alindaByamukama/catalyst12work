Object Oriented Programming

A class is a blue print of an object and an object is an instance of a class (they have properties). The five main concepts you need to know about are:

  1. Inheritance
  2. Abstraction
  3. Polymorphism
  4. Encapsulation
  5. Instances

You can acces the properties of an object using dot notation. An example of a class is "console.log()" --> console is a class and log() function is an object of console. Methods have behaviours and methods --> A method defines what it does to itself or to others.
